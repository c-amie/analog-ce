# Yearly Report
Отчет об активности в течение каждого года.
# Quarterly Report
Отчет об активности в течение каждого квартала.
# Monthly Report
Отчет об активности в течение каждого месяца.
# Weekly Report
Отчет об активности в течение каждой недели.
# Daily Report
Отчет об активности в течение каждого дня.
# Hourly Report
Отчет об активности в течение каждого часа.
# Quarter-Hour Report
Отчет об активности в течение каждых 15 минут.
# Five-Minute Report
Отчет об активности в течение каждых 5 минут.
# Daily Summary
Отчет об активности в течение каждого дня недели, суммированной по всем неделям, представленным в отчете.
# Hourly Summary
Отчет об активности в течение каждого часа дня, суммированной по всем дням, представленным в отчете.
# Hour of the Week Summary
Отчет об активности в течение каждого часа недели, суммированной по всем неделям, представленным в отчете.
# Quarter-Hour Summary
Отчет об активности в течение каждых 15 минут дня, суммированной по всем дням, представленным в отчете.
# Five-Minute Summary
Отчет об активности в течение каждых 5 минут дня, суммированной по всем дням, представленным в отчете.
# General Summary
Этот отчет содержит всю доступную информацию.
# Request Report
Этот отчет показывает список файлов на сайте.
# Redirection Report
Этот отчет показывает список файлов, которые послужили причиной перенаправления запроса. 
# Failure Report
Этот отчет показывает файлы, которые явились причиной ошибок, например, те файлы, которые не были найдены.
# File Type Report
Этот отчет показывает расширения затребованных файлов.
# Directory Report
Этот отчет показывает каталоги, из которых были затребованы файлы.
# Host Report
Этот отчет показывает список машин, с которых были затребованы файлы.
# Host Redirection Report
Этот отчет показывает список машин, запросы с которых были перенаправлены к другому файлу.
# Host Failure Report
Этот отчет показывает список машин, которые сделали ошибочный запрос.
# Domain Report
Этот отчет показывает страны (домены), машины которых делали запрос.
# Referrer Report
Этот отчет показывает источники, откуда люди перешли на наш сайт.
# Referring Site Report
Этот отчет показывает сервера, с которых люди зашли на наш сайт.
# Redirected Referrer Report
Этот отчет показывает источники, вызвавшие перенаправление запроса.
# Failed Referrer Report
Этот отчет показывает источники, которые содержат недействительные ссылки на наш сайт.
# Browser Report
Этот отчет показывает броузеры, используемые посетителями.
# Virtual Host Report
Этот отчет показывает посещаемость каждого из виртуальных доменов системы.
# Virtual Host Redirection Report
Этот отчет показывает перенаправленные запросы к каждому виртуальному домену.
# Virtual Host Failure Report
Этот отчет показывает ошибочные запросы к каждому из виртуальных доменов.
# User Report
Этот отчет показывает пользователей, которые затребовали файлы, если пользователя удалось идентифицировать.
# User Redirection Report
Этот отчет показывает пользователей, которые были перенаправлены к другому файлу.
# User Failure Report
Этот отчет показывает пользователей, которые произвели ошибочный запрос.
# Organisation Report
Этот отчет показывает организации, которые произвели запрос.
# Search Query Report
Этот отчет показывает, какие запросы произвели пользователи в поисковых машинах, чтобы найти наш сайт.
# Search Word Report
Этот отчет показывает, какие слова использовали пользователи, чтобы найти наш сайт.
# Internal Search Query Report
Этот отчет показывает, какие запросы использовали пользователи в скриптах на нашем сайте.
# Internal Search Word Report
Этот отчет показывает, какие слова использовали пользователи в скриптах на нашем сайте.
# Browser Summary
Этот отчет показывает, какие производителей броузеров пользователей.
# Operating System Report
Этот отчет показывает операционные системы, используемые пользователями.
# File Size Report
Этот отчет показывает размер файлов.
# Status Code Report
Этот отчет показывает HTTP коды всех запросов.
# Processing Time Report
Этот отчет показывает время, потребовавшееся для выполнения успешных запросов.
