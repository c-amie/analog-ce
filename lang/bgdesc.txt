# Report descriptions file for analog 6.0.
#
# This file contains descriptions of each report. Lines starting with a
# hash (#) are comments. Otherwise, each report must take EXACTLY ONE LINE.
# Bulgarian translation 25.I.2001 L.Georgiev <lucho@mbox.digsys.bg>
# Windows-1251 (Windows / X) version
#
# Yearly Report
Този отчет опис активността за всяка година.
# Quarterly Report
Този отчет описва активността за всяко тримесечие.
# Monthly Report
Този отчет описва активността за всеки месец.
# Weekly Report
Този отчет описва активността за всяка седмица.
# Daily Report
Този отчет описва активността за всеки ден.
# Hourly Report
Този отчет описва активността за всеки час.
# Quarter-Hour Report
Този отчет описва активността за всеки четвъртчасов период.
# Five-Minute Report
Този отчет описва активността за всеки петминутен период.
# Daily Summary
Този отчет описва общата активност за всеки ден от седмицата, сумирана за всички седмици на отчета.
# Hourly Summary
Този отчет описва общата активност за всеки час на деня, сумирана за всички дни на отчета.
# Hour of the Week Summary
Този отчет описва общата активност за всеки час от седмицата, сумирана за всички седмици на отчета.
# Quarter-Hour Summary
Този отчет описва общата активност за всеки четвъртчасов период от деня, сумирана за всички дни на отчета.
# Five-Minute Summary
Този отчет описва общата активност за всеки петминутен период от деня, сумирана за всички дни на отчета.
# General Summary
Този отчет съдържа цялостна статистика.
# Request Report
Този отчет описва файловете на сайта.
# Redirection Report
Този отчет описва файловете, които са предизвикали пренасочване на заявките към друг файл. (Обикновено директории с липсваща последна наклонена черта или CGI-скриптове, предизвикали пренасочване.)
# Failure Report
Този отчет описва файловете, предизвикали грешки, например файлове, които не са намерени.
# File Type Report
Този отчет описва разширенията на заявените файлове.
# Directory Report
Този отчет описва директориите, от които са били заявени файлове. (Числата за всяка директория включват всички нейни поддиректории.)
# Host Report
Този отчет описва компютрите, които са заявили файлове.
# Host Redirection Report
Този отчет описва компютрите, които са били пренасочени към друг файл.
# Host Failure Report
Този отчет описва компютрите, които са се натъкнали на сгрешени заявки.
# Domain Report
Този отчет описва страните, в които са били компютрите, заявили файлове.
# Referrer Report
Този отчет описва отпращачите (откъдето хората са следвали връзките или страници, които са включвали изображения от този сайт).
# Referring Site Report
Този отчет описва от кои сървери хората са следвали връзки.
# Redirected Referrer Report
Този отчет описва отпращачите, които са предизвикали пренасочени заявки.
# Failed Referrer Report
Този отчет описва отпращачите, съдържащи невалидни връзки към сайта.
# Browser Report
Този отчет описва браузерите, използвани от посетителите.
# Virtual Host Report
Този отчет описва активността на всяка от виртуалните области на сайта.
# Virtual Host Redirection Report
Този отчет описва пренасочените заявки към всяка от виртуалните области.
# Virtual Host Failure Report
Този отчет описва сгрешните заявки към всяка от виртуалните области.
# User Report
Този отчет описва потребителите, които са заявили файлове, ако самоличността им е била установена или могат да бъдат идентифицирани от "късметчета".
# User Redirection Report
Този отчет описва потребителите, които са били пренасочени към друг файл.
# User Failure Report
Този отчет описва потребителите, които са се натъкнали на сгрешени заявки.
# Organisation Report
Този отчет описва the организациите, в които са били компютрите, заявили файлове.
# Search Query Report
Този отчет описва кои фрази са използвали хората в търсещите машини, за да намерят сайта.
# Search Word Report
Този отчет описва кои думи са използвали хората в търсещите машини, за да намерят сайта.
# Internal Search Query Report
Този отчет описва кои фрази са използвали хората в скриптове в рамките на сайта.
# Internal Search Word Report
Този отчет описва кои думи са използвали хората в скриптове в рамките на сайта.
# Browser Summary
Този отчет описва типовете браузери на посетителите.
# Operating System Report
Този отчет описва операционните системи, използвани от посетителите.
# File Size Report
Този отчет описва размерите на файловете.
# Status Code Report
Този отчет описва HTTP-кода на състоянието на всички заявки.
# Processing Time Report
Този отчет описва времената, които са били необходими за обработка на успешните заявки.
